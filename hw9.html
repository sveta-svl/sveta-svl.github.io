SVG Files in CNIT 131A

For this assignment I created SVG graphics using hand-coded XML/SVG markup directly in the HTML file. Each shape is defined with SVG elements such as <rect>, <circle>, and <polygon>. The gradient shape uses a <linearGradient> defined in the SVG <defs> section and applied via the fill attribute. No external tools were used — all SVG was written by hand.
